SCRIPTS DIRECTORY
  scripts/db_migrate.py
    - Convenient CLI wrapper for Alembic commands
    - Provides high-level commands like status, upgrade, downgrade, reset
    - Fallback database initialization from model metadata
    - Drop-all functionality for development
    Commands:
      status              - Show current migration state
      upgrade [target]    - Apply migrations up to target
      downgrade [target]  - Downgrade migrations to target
      reset               - Reset database to initial state (dev only)
      sql [target]        - Generate SQL for migrations
      history             - Show migration history
      current             - Show current migration
      revision <msg>      - Create new migration
      init-fallback       - Initialize from model metadata
      drop-all            - Drop all tables (dev only)
      help                - Show help message
    Path: /sessions/stoic-determined-tesla/mnt/wifi/CommClient-Server/scripts/db_migrate.py

USAGE SUMMARY
================================================================================

1. Install dependencies:
   pip install -r requirements.txt

2. Apply initial migration:
   alembic upgrade head

3. Verify:
   alembic current

4. Make model changes, then:
   alembic revision --autogenerate -m "description"
   alembic upgrade head

5. For production, generate SQL:
   alembic upgrade head --sql > migration.sql
