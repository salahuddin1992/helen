1. app/services/metrics_service.py (88 lines)
   ─────────────────────────────────────────
   - MetricsService class for lightweight in-memory metrics collection
   - Tracks: messages_sent_total, calls_initiated_total, files_uploaded_total,
     socket_connections_total, api_requests_total
   - Methods:
     * async increment(metric_name, value=1)
     * async get_all() → dict[str, int]
     * get_uptime() → float
     * async reset()
   - Thread-safe via asyncio.Lock
   - Singleton: metrics_service

2. app/services/backup_service.py (254 lines)
   ───────────────────────────────────────────
   - BackupService class for SQLite database backup/restore operations
   - Backup storage: data/backups/commclient_backup_YYYYMMDD_HHMMSS.db
   - Methods:
     * async create_backup() → str
     * async list_backups() → list[dict]
     * async restore_backup(backup_name: str) → bool
     * async delete_backup(backup_name: str) → bool
     * async auto_cleanup(keep_count: int = 10) → int
     * async get_db_size() → int
   - Path traversal protection against malicious backup names
   - Creates protective backup before restore
   - Thread-safe via asyncio.Lock
   - Singleton: backup_service

3. app/services/admin_service.py (324 lines)
   ─────────────────────────────────────────
   - AdminService class for server diagnostics and user management
   - System metrics via optional psutil (gracefully degrades if unavailable)
   - Methods:
     * async get_server_stats() → dict[str, Any]
     * async get_active_calls() → list[dict]
     * async kick_user(user_id: str) → bool
     * async ban_user(db: AsyncSession, user_id: str) → bool
     * async unban_user(db: AsyncSession, user_id: str) → bool
     * async get_audit_log(limit: int = 100) → list[dict]
     * async cleanup_expired_sessions(db: AsyncSession) → int
     * async cleanup_orphaned_files(db: AsyncSession) → int
   - Helper: get_lan_ip() → str (network utility)
   - Singleton: admin_service

4. app/api/routes/admin.py (489 lines)
   ──────────────────────────────────
   - FastAPI router with 11 authenticated endpoints
   - Prefix: /api/admin
   - All endpoints require valid Bearer token
   - Endpoints:
     GET    /stats                    — Server statistics dashboard
     GET    /active-calls             — List active calls
     POST   /kick/{user_id}           — Force disconnect user
     POST   /ban/{user_id}            — Ban user (soft)
     POST   /unban/{user_id}          — Unban user
     POST   /cleanup/sessions         — Clean expired JWT sessions
     POST   /cleanup/files            — Clean orphaned files
     GET    /backups                  — List backups
     POST   /backups                  — Create backup
     POST   /backups/{name}/restore   — Restore backup
     DELETE /backups/{name}           — Delete backup
   - Full error handling with HTTP status codes
   - Security audit logging on all operations

INTEGRATION STEPS
