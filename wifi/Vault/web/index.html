⚠ تحذير أمني

  • Plaintext لا يُكشف إلا حين الضغط على "كشف" — وكل عملية كشف تُسجَّل في server log
  • التغيير الفوري — الرمز الجديد ساري قبل ما تنتهي هذه الـrequest
  • الرموز الضعيفة (مثل 123456789) مقبولة تقنياً لكن ليست توصية
  • كل جلسات الطبقة المتعلقة تُبطَل عند التغيير (admin أو vault)
